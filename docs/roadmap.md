# Roadmap

## Foundation

- Monorepo, local infrastructure, quality tooling, and continuous integration
- Stable API contracts and authentication strategy
- Observability, error reporting, and deployment environments

## Generation platform

- Versioned site specification schema
- Provider-neutral AI gateway with evaluation and cost controls
- Idempotent queue workflows and generation audit trail
- Deterministic renderer with preview environments

## Publishing

- Asset pipeline and managed deployment targets
- WordPress publishing adapter
- Approval, revision, rollback, and collaboration workflows

Roadmap items describe direction only and are not implemented in the initial scaffold.
